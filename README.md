# 연구실 운영 Wiki

랩 구성원용 운영 위키입니다. 공개 주소에 접속해 비밀번호를 입력하면 내용을 볼 수 있습니다.

- 주소: https://hongkiyoo.github.io/boom-wiki/
- 비밀번호: 운영조교 / PI 에게 문의

> 이 저장소에는 **암호화된 결과물(`index.html`)만** 들어 있습니다. GitHub Pages 가 이 파일을 서빙합니다.

## 편집은 어디서 하나요?

내용 편집과 운영은 별도의 **비공개 소스 저장소 `boom-wiki-src`** 에서 합니다.
거기서 원본을 고치면 GitHub Actions 가 자동으로 암호화하여 이 저장소로 배포합니다.
편집 권한이 필요하면 PI / 운영조교에게 문의하세요. (운영 방법은 `boom-wiki-src` 의 README 참고)

> 이 저장소의 `index.html` 은 자동 배포로 갱신되므로 **직접 수정하지 마세요.**
